airve.github.com uses GitHub Pages as a CDN to serve cachable versions of common opensource scripts. I set this up so that I'd have CDN links for my plugins and for other resources I use frequently. Feel free to use any of the links below. The links serve the latest version and get cachebusted when updated. To add a library, push to [the repo](https://github.com/airve/airve.github.com)'s master branch, or suggest it here, or hit me up @ryanve. Also checkout cdnjs.com.
