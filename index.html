airve.github.io uses GitHub Pages as a CDN to serve cachable versions of common opensource scripts. I set this up so that I'd have CDNish links for my plugins and for other resources I use frequently. Feel free to use the links below as you would from a CDN. The links serve the latest version and get cachebusted when updated or every 2 days. To add a library, push to the repo's master branch, suggest it here, or ask @ryanve.
